# Outreach B2B Platform - Command Centre Dashboard

## Tech Stack
- **Framework**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **State Management / Data Fetching**: @tanstack/react-query
- **Routing**: react-router-dom
- **Charts**: recharts
- **Icons**: lucide-react

## File Structure
- `src/components`: Reusable UI components and feature-specific modules
- `src/pages`: Main view components
- `src/services`: API integration
- `src/types`: TypeScript definitions

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Development Guidelines
- Follow the design system in `tailwind.config.js`.
- Use specific components from `src/components/ui` for consistency.
